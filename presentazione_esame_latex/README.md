# Canovaccio LaTeX per la presentazione d'esame

Questa cartella contiene una bozza **Beamer** in italiano (`main.tex`) costruita sulla repository, con focus su:

1. introduzione matematica e definizioni;
2. esperimento "Transazioni" (wallet, genesis, nodi, tx, mining, state);
3. esperimento "Attacco Weak Nonce Reuse" (setup, vulnerabilità, exploit, recovery).

## Compilazione

Se nel sistema è installato LaTeX:

```bash
cd presentazione_esame_latex
pdflatex main.tex
```

## Nota didattica
